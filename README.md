# Nuestra-aventura-en-el-Bosque-Encantado
Proyecto digital para que el alumnado de primaria cree una revista interactiva sobre el viaje al Bosque Encantado 4D. Los alumnos organizarán información, redactarán itinerarios, crearán mapas, fotos, collages y mini-reportajes, e incluirán vocabulario en inglés.
